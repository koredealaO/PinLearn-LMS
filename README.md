\# PinLearn LMS



A modern AI-powered Learning Management System built with Flask.



\## Version



v0.1.0-alpha



\## Technology Stack



\- Flask

\- SQLAlchemy

\- Bootstrap 5

\- SQLite (Development)

\- PostgreSQL (Production)



\## Installation



\### Clone



```bash

git clone <repository-url>

```



\### Create Virtual Environment



```bash

python -m venv .venv

```



Windows



```bash

.venv\\Scripts\\activate

```



Linux/macOS



```bash

source .venv/bin/activate

```



\### Install Dependencies



```bash

pip install -r requirements.txt

```



\### Configure Environment



Copy



```

.env.example

```



to



```

.env

```



Edit the values.



\### Run



```bash

python run.py

```



Open



```

http://127.0.0.1:5000

```
